# 📈 TradeX - Stock Trading Platform

TradeX is a full-stack MERN-based stock trading platform inspired by modern trading applications like Dhan. The project focuses on building a scalable, production-ready architecture while implementing real-world trading features.

## 🚀 Features

### ✅ Completed

* JWT Authentication
* Secure HTTP-only Cookie Authentication
* User Registration & Login
* Protected Routes
* Input Validation using Zod
* Feature-Based Backend Architecture
* MongoDB Atlas Integration
* Global Error Handling
* Rate Limiting
* Health Check API

### 🚧 In Progress

* Live Market Data
* Stock Search
* Watchlist
* Portfolio Management
* Paper Trading
* Real-Time Price Updates
* TradingView Charts
* IPO Module
* Market News
* Option Chain
* Technical Indicators
